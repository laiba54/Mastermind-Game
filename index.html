Mastermind Game

Game Rules

1. The codemaker creates a secret code.

2. The codebreaker tries to guess the secret code within a certain number of turns.

3. The codemaker selects a sequence of 4 pegs, each peg can be one of 6 colors, duplicates are allowed.

4. The codebreaker places a sequence of 4 colored pegs in the current row and the codemaker provides feedback with small pegs: Black peg: Correct color and correct position. White peg: Correct color but wrong position. No peg: Color not in the code.

5. The codebreaker makes guesses and receives feedback until the code is guessed correctly or the codebreaker runs out of turns.

6. The codebreaker wins if they guess the code within the allowed number of turns. The codemaker wins if the codebreaker does not guess the code within the allowed number of turns.
